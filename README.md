# Frontend-Website-Design
I created a frontend website design on the topic "Vikasit Bharat"
This project showcases a complete frontend webpage design conceptualized and created from scratch in Figma, and then developed using HTML and CSS.

✨ Design Process

All UI elements, vectors, and visual assets were self-created in Figma.

The layout, typography, color system, and components were thoughtfully structured to ensure visual consistency and user experience.

The original Figma design has been exported as a PDF and included in this repository for reference.

💻 Development

The static webpage is implemented using clean and structured HTML.

Styling and layout are handled entirely through CSS.

The final output closely matches the original Figma design to maintain design-to-code accuracy.

📁 Repository Structure

Design.pdf – Figma design export

index.html – Webpage structure

style.css – Styling and layout

🚀 Purpose

This project demonstrates my ability to:

Create custom UI elements and vector graphics

Design complete layouts in Figma

Convert design systems into responsive frontend code

Maintain pixel-level design consistency
